## 📌 Title
FastTerminal for Java — High-Performance JLine-Free True-Color Terminal Engine

## 📝 Description
FastTerminal is a high-performance, JLine-free terminal graphics and TUI engine for Java. 
Engineered for high-refresh rates (locked 120 FPS), 24-bit True Color, and native JNI console size tracking, it provides a lightweight virtual cell buffer substrate completely free of standard Java AWT/Swing or heavy terminal wrappers.

### 🚀 Key Features:
- **Locked 120 FPS Refresh Rate**: Seamless rendering with ultra-low latency JNI console dimension tracking.
- **🎨 24-Bit True Color**: Support for direct RGB escape codes for foreground and background customization.
- **🌐 UTF-32 Codepoint Grids**: Zero-copy primitive integer cell buffers that eliminate Emoji/Unicode surrogate split bugs (e.g. 🚀, 🌈, ⚡).
- **🛡️ State-Minimized Compositor**: Bandwidth-saving compositor only emits ANSI escape codes when color states change.
- **🎹 Native Windows Integration**: Query console screen buffer sizing (`csbi.dwSize.X`) and modes directly via C++ JNI DLL layers.
- **🎹 Zero Dependencies**: Integrates natively with `FastKeyboard` and `FastMouse` to build responsive, premium command-line visual dashboards.

### 🔥 Use Cases:
- Real-time terminal games, visual simulations, and sines.
- Native, responsive full-screen TUI dashboards and status applications.
- Premium developer developer toolkits and interactive CLI builders.
